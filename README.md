# Globle

A centralized repository for Claude AI coding profiles, skills, agents, and workflows.

## Structure

```
Globle/
├── CLAUDE.md              # Main Claude instructions & preferences
├── agents/                # Agent definition files
│   └── workflows/         # Workflow definitions for agents
├── skills/                # Reusable skill modules
└── resources/             # Shared resources, templates, and assets
```

## Usage

Clone this repo and reference its contents from your projects to maintain a consistent Claude coding profile across all your work.

```bash
git clone https://github.com/coroambra/Globle.git
```
