# Workflows

This folder contains workflow definitions — step-by-step instructions for common tasks.

## How to Add a Workflow

1. Create a new `.md` file (e.g., `deploy.md`)
2. Add YAML frontmatter with a description
3. Write clear, numbered steps

## Workflow Template

```markdown
---
description: Short description of what this workflow does
---

1. First step
2. Second step
3. Third step
```
