# Agents

This folder contains agent definitions. Each agent is a markdown file that defines a specialized AI assistant persona.

## How to Add an Agent

1. Create a new `.md` file in this folder (e.g., `code_reviewer.md`)
2. Define the agent's role, capabilities, and instructions
3. Reference it from your projects as needed

## Agent Template

```markdown
# Agent Name

## Role
Describe what this agent specializes in.

## Instructions
- Specific behaviors and rules
- Tools and techniques to use
- Boundaries and limitations

## Context
Any background knowledge the agent should have.
```
